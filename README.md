# Diamante de X

## Funcionalidade

- **Validação de Entrada**: O diamanteX garante que apenas números válidos sejam aceitos.
- **Tratamento de número par**: Valida se o número digitado é par.

---

## Como usar

1. Clone o repositório ou baixe o código fonte do diamanteX.
2. Abara o terminal ou prompt de comando e navegue até a pasta raiz do diamanteX.
3. Utilize o comando abaixo para restauraras dependências do projeto.
```
dotnet restore
```

4. Em seguinda, compile a solução utilizando o comando:
```
dotnet build --configuration Release
```

5. Para executar o projeto compilado em tempo real
```
dotnet run --project DiamenteX.ConsoleApp
```

6. Para executar o arquivo compilado, navegue ate a pasta `./DiamanteX.ConsoleApp/bin/Release/net8.0/` e execute os arquivos.

## Requesitos 

- .NET SDK (recomendado .NET 5.0 ou superior) para compilação e execução do projeto.
